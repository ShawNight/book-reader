## 🎯 三个主要标签页

### 1️⃣ 书架（当前）
- **功能：** 查看收藏的书籍
- **右上角：** 🔍搜索按钮
- **书籍信息：** 封面、书名、作者、阅读进度
- **操作：** 点击阅读、左滑删除

### 2️⃣ 书源
- **功能：** 管理书源
- **右上角：** ➕导入书源按钮
- **列表：** 显示所有书源
- **状态：** ✓已启用 / ✗已禁用

### 3️⃣ 设置
- **功能：** 应用设置
- **选项：** 主题、字体、关于
- **信息：** 版本号

---

## 📋 操作指南

### 查看书源页面
```
点击底部【📖 书源】标签
```

### 导入书源
```
书源标签 → 右上角➕ → 选择书源.json
```

### 搜索小说
```
书架标签 → 右上角🔍 → 输入"斗破"
```

### 加入书架
```
搜索结果 → 点击右侧⊕按钮
```

### 阅读书籍
```
书架 → 点击书籍 → 选择章节 → 阅读
```
