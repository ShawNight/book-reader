## 🎨 设计特点

### 配色方案
- **主题色：** 蓝色（Material Design 3）
- **背景色：** 米黄色（护眼，#F5F5DC）
- **文字色：** 深灰/黑色
- **强调色：** 绿色（✓已启用）

### 字体
- 标题：16-20px，加粗
- 正文：18px（阅读舒适）
- 辅助：12-14px，灰色

### 交互
- 点击：按钮、列表项
- 滑动：PageView左右翻页
- 长按：（待实现）

---

## 📐 尺寸规范

- **底部导航栏：** 80px高
- **AppBar：** 56px高（默认）
- **列表项：** 72px高（3行）
- **边距：** 16px
- **圆角：** 4px（Material 3）

## 🔄 导航流程

```
书架 → [搜索] → 搜索结果 → [点击书籍] → 章节列表 → [点击章节] → 阅读
  ↓
书源 → [导入] → 书源列表
  ↓
设置 → 主题/字体/关于
```
